# 1. Pull/build new image
docker-compose build fastapi-api

# 2. Restart only the app containers (DB stays untouched)
docker-compose up -d --no-deps fastapi-api workers nginx

# 3. Run any new migrations (adds tables/columns, never drops)
docker-compose exec fastapi-api alembic upgrade head

Never run docker-compose down -v — the -v flag deletes volumes (your data). Use docker-compose down without -v.

Back up before deploy:
docker exec iotaps-postgres pg_dump -U iotaps iotaps > backup_$(date +%Y%m%d).sql
